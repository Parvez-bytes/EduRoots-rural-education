How to Play

  1. Click on input bulbs A and B to toggle them ON/OFF
  2. Select a logic gate from the options
  3. Observe how the output changes based on inputs and selected gate
  4. Click "Check Circuit" to verify if you've completed the challenge
  5. Earn points and advance to higher levels with more complex challenges

* Note: NOT gate only uses Input A
