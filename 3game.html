How the Game Works:

  • Each level generates random coordinates based on difficulty
  • Questions are never the same twice - great for repeated practice!
  • Levels 1-5: Positive integers only (Quadrant I)
  • Levels 6-10: Integers in all four quadrants
  • Levels 11-15: Decimal numbers in all quadrants
  • Use the "New Question" button to get a different point at the same difficulty level
